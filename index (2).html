1. Pengertian OOP

Object Oriented Programming (OOP) adalah paradigma pemrograman yang berorientasi pada objek. Paradigma ini menekankan penggunaan class dan object dalam pengembangan program.

Dengan OOP, kode menjadi lebih modular, mudah dikelola, dan dapat digunakan kembali. Konsep penting: Encapsulation, Inheritance, Polymorphism, Abstraction.

2. Class dan Object

Sebuah class adalah blueprint untuk membuat object. Object adalah instansi dari class.


class Car {
  String? model;
  String? color;

  void start() {
    print('Car started!');
  }

  void stop() {
    print('Car stopped!');
  }
}

3. Constructor

Constructor adalah method khusus untuk inisialisasi object saat dibuat.


class Car {
  String? model;
  String? color;

  Car(this.model, this.color);

  void start() {
    print('Car started!');
  }
}

4. Validasi Data

Validasi untuk memastikan atribut memenuhi aturan. Contoh: IPK harus 0.0 – 4.0


class Mahasiswa {
  String nama;
  String nim;
  double ipk;

  Mahasiswa(this.nama, this.nim, this.ipk) {
    if (ipk < 0.0 || ipk > 4.0) {
      throw ArgumentError('IPK harus 0.0 - 4.0');
    }
  }
}

5. Encapsulation

Encapsulation membungkus atribut dan method agar akses terkendali. Biasanya atribut private diakses lewat getter/setter.


class Person {
  String _name;
  int age;

  Person(this._name, this.age);

  void setName(String name) {
    _name = name;
  }

  String getName() {
    return _name;
  }
}

6. Named Parameter Constructor

Named Parameter Constructor memudahkan pengisian atribut object dengan nama parameter jelas.


class Produk {
  final String id;
  final String nama;
  final double harga;
  final int stok;
  final String kategori;

  Produk({
    required this.id,
    required this.nama,
    required this.harga,
    this.stok = 0,
    this.kategori = 'Umum',
  });
}
